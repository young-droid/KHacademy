        form 태그 : 내부에 작성된 input 태그의 값을 지정된 서버 또는 페이지로
                    제출(전달)하는 역할

        *이 때, 어떤 input태그에 작성된 값이 제출된건지 알 수 있도록
        input 태그에 name 속성을 반드시 작성해야 한다.

        form 태그 속성
        -action : input 태그의 값을 전달핧 서버 또는 페이지 주소를 작성하는 속성
        -method : 데이터 전달 방식을 지정하는 속성(get/post)
