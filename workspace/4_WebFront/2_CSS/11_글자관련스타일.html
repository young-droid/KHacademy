color : 글자 색을 지정하는 속성

        선택자{ color : 색상명(영문) | 16진수 숫자 RGB(#fff, #ffffff)
                        | rgb(255, 255, 255) | rgba(255, 255, 255, 1)
                        | hsl(360, 100, 100) | hsla(360, 100, 100, 1)}

        * rgba에서 a는 투명도(0 : 투명, 1 : 불투명)
        * hsl(색조, 채도, 밝기)
        * 6개의 색상 지정 방법은 색과 관련된 모든 css 속성에서 사용 가능
  • 색상명으로 지정
